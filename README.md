# KumulAirs Chatbot - PNG Air AI Assistant 🇵🇬

**AI-powered customer service chatbot for PNG Air / Air Niugini**  
**Auto-detects Tok Pisin & English | Covers all 22 Provinces**
## ✈️ Key Features
- **Bilingual Support**: Automatically replies in Tok Pisin or English
- **22 Province Coverage**: POM, LAE, RAB, MDG, HGU, BUA, KVG + all others
- **PX Route Logic**: Daily flights POM-LAE, POM-RAB, POM-MDG, etc.
- **Smart Intent Handling**: Booking, pricing, baggage rules
- **Baggage Rules**: Economy 16kg | Business 30kg | Hand carry 7kg
- **Anger Management**: Professional fallback to customer service
- **Customer Service**: Directs complex queries to 180 3444
## 🛠️ Built With
- **Python 3** + **Flask** REST API
- **Regex** language detection
- **Deploy Ready**: Railway, Render, 
## 🚀 Quick Start
```bash
pip install flask
python app.py
```
